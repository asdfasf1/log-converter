기본 마크다운 규칙 (자동 적용)
• **볼드** → <strong> 태그
• *이탤릭* → <i> 태그
• "대사" → 색상/굵기 강조
• '속마음' → 속마음 스타일 강조
• > 인용 → 인용구 스타일 적용
• >> / >>> 인용 → 좌/우 정렬 인용구 적용
• #, ##, ### → 제목(H1~H3)
• --- → 구분선
• 빈 줄 → 문단 구분
커스텀 치환어 추가

예: 패턴: \[(.*?)\] / 치환: <strong>$1</strong>
